# Abang Tahu Bulat (Discord Bot)

Abang Tahu Bulat adalah project pembuatan bot discord kecil-kecilan

## Installation

```bash
npm install
```

## Compile

```bash
node my-bot.js
```
## Usage
Get a quote
```bash
bang quote
```

Get a cat image
```bash
bang kucing
```

Play Ular Tangga
```bash
bang ular-tangga
```
